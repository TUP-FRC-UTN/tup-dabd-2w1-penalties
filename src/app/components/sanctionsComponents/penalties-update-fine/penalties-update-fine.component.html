Actualizar Precio Multa

Aquí se puede realizar la modificación del importe de una multa. La multa incluye la siguiente información:

  • Motivo: Motivo por el cuál se registró la multa. El campo no se puede modificar.
  • Fecha: Fecha en la cuál se registró la multa. El campo no se puede modificar.
  • Descripción: Descripción que acompaña a la razón de la multa. El campo no se puede modificar.
  • Importe: Aquí estará el importe con el cuál se creó la multa. El mismo puede ser modificado en base a la multa correspondiente.

Recuerde que cambiar el importe no es una decisión menor; asegúrese de estar seguro antes de realizar el cambio.
